# 3ML3 Workspace

This workspace is organized by course material type:

- `HW1` through `HW5`: homework notebooks, reports, and study guides
- `Exercises`: chapter exercise notebooks
- `chapter_8_datasets`, `chapter_10_datasets`, `chapter_8_library`: shared datasets and helper code used by the notebooks
- `MLRPlotting`: local plotting helper package
- `mlr-assets`: external nested repository kept separate from the main workspace repo

Start with `HOMEWORK_STUDY_INDEX.md` if you want the study sequence across the homework folders.
